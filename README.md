## State Management

This project uses **Redux Toolkit** for global state. The store is configured in `store/index.ts` with a sample `counter` slice. Add more slices as needed and use the `useAppDispatch`/`useAppSelector` hooks from `store/hooks.ts`.

### Example component

A `components/ReduxCounter.tsx` is included to demonstrate usage:

```tsx
"use client";
import { useAppDispatch, useAppSelector } from "../store/hooks";
import { increment, decrement } from "../store/counterSlice";

function ReduxCounter() {
  const count = useAppSelector((s) => s.counter.value);
  const dispatch = useAppDispatch();
  return (
    <div>
      <button onClick={() => dispatch(decrement())}>-</button>
      <span>{count}</span>
      <button onClick={() => dispatch(increment())}>+</button>
    </div>
  );
}
```

Import and render the component anywhere under `app/` (client component) to see state updates in action.
